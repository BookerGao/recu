# 票迹 / recu

票迹 App 的官方支持站点，用于 App Store Connect 的隐私政策和支持链接。

## Pages

- `index.html`: 支持中心入口
- `privacy.html`: App Store Connect 隐私政策
- `support.html`: App Store Connect 支持与反馈

## App Store Connect URLs

After enabling GitHub Pages for this repository, use:

- Privacy Policy URL: `https://bookergao.github.io/recu/privacy.html`
- Support URL: `https://bookergao.github.io/recu/support.html`

## Enable GitHub Pages

1. Open the repository on GitHub.
2. Go to `Settings` > `Pages`.
3. Under `Build and deployment`, choose `Deploy from a branch`.
4. Select branch `main` and folder `/ (root)`.
5. Save and wait for GitHub to publish the site.
