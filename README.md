# Ticca

Ticca App 的官方支持站点，用于 App Store Connect 的隐私政策和支持链接。

## Pages

- `index.html`: 支持中心入口
- `privacy.html`: App Store Connect 隐私政策
- `support.html`: App Store Connect 支持与反馈
- `ticca/privacy.html`: Ticca 隐私政策路径
- `ticca/support.html`: Ticca 支持与反馈路径

## App Store Connect URLs

After enabling GitHub Pages for this repository, use:

- Privacy Policy URL: `https://bookergao.github.io/ticca/privacy.html`
- Support URL: `https://bookergao.github.io/ticca/support.html`

If this repository is still served as a project page named `recu`, GitHub Pages may publish it under `https://bookergao.github.io/recu/`. In that case, rename the repository to `ticca` or move these files to a user/organization Pages root so the `/ticca/` URLs resolve.

## Enable GitHub Pages

1. Open the repository on GitHub.
2. Go to `Settings` > `Pages`.
3. Under `Build and deployment`, choose `Deploy from a branch`.
4. Select branch `main` and folder `/ (root)`.
5. Save and wait for GitHub to publish the site.
